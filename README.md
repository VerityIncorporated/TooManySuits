# TooManySuits

A [More Suits](https://thunderstore.io/c/lethal-company/p/x753/More_Suits/) addon that adds pages to the suit rack.

![screenshot](https://i.imgur.com/uuvyEjo.png)

## Default Controls

- Next page: `n`
- Back page: `b`
